# The Atelier

> **Wear the look. Share the thread.**

The Atelier is a creator-focused FFXIV Dalamud plugin for discovering, publishing, saving, and applying glamour looks through Aetherweave.

Browse community outfits, save favorites to your Wardrobe, review required mods, apply compatible Glamourer appearances, and publish your own looks with previews, categories, tags, and dependency links.

## Highlights

- Online glamour Showcase
- Personal Wardrobe
- Aetherweave outfit sharing
- Glamourer appearance capture
- Creator profiles and portfolios
- Drafts and autosave
- Publication review status
- SFW and opt-in NSFW filtering
- VirusTotal checks for public mod links
- External-link safety confirmations
- Discord-backed creator authentication

The Atelier does not redistribute mod files. Users install dependencies from their original sources and remain responsible for reviewing external links before continuing.

`https://raw.githubusercontent.com/rubyblaire/TheAtelier/main/pluginmaster.json`

`/atl`
`/atelier`
